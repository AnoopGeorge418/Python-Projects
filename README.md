## **Getting Started**

1. Clone this repository:
   ```
   git clone https://github.com/AnoopGeorge418/Python-Projects.git
   ```
2. Navigate to the project you're interested in:
   ```
   cd Python-Projects/[Project Name]
   ```
3. Run the main.py
   ```
   python main.py
   ```
